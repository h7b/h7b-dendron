# Functionplot

- GitHub: https://github.com/leonhma/obsidian-functionplot
- Thoughts:
    - A plugin for displaying mathematical graphs in Obsidian
- demo
    - This example code will be rendered as the picture below
    ```` 
    ```functionplot
    ---
    title: Graph
    disableZoom: false
    bounds: [-10, 10, -10, 10]
    grid: true
    xLabel: x
    yLabel: y
    ---
    
    f(x) = x^2
    g(x) = 0.5*x^3+x^2-44
    ```
    ````
